# 🎧 AI DJ: Autonomous Innovation Pipeline

**AI DJ** is a fully autonomous system designed to discover, invent, and validate next-generation music transitions. It leverages a multi-agent "Brain" to research techniques, mutate them into novel innovations, and pre-validate them before user testing.

## 🚀 Core Mission
The goal of this project is to remove the human "middle-man" from the DJ transition workflow. The system identifies track pairs, analyzes their DNA (BPM, Key, Energy), and applies advanced techniques—some learned from YouTube masters, and some invented by the system's own mutation logic.

## 🛠️ Quick Start
1. **Library**: Add `.mp3` files to `data/library`.
2. **Run**: Execute `python mobile_tester.py`.
3. **Test**: Use the public link to rate the generated mixes.

## 🤖 Continuous Innovation Loop
The system runs a background "Factory" that:
1. Researches new techniques via Google.
2. Generates 10 candidate "Invention" mixes.
3. Ranks them using the **Validation Agent**.
4. Sends the **Top 1** candidate to the user for final approval.
